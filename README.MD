功能说明：
	服务器已经有计算千万亿次的地址计算划分为，当前程序使用N卡RTX2070 速度600M。
	程序向服务器发起计算区间请求，每次数据计算区间为1000亿，RTX2070耗时160秒左右。

操作步骤：
	1.安装显卡驱动，建议更新到N卡官网最新版。
	2.安装VS2017 （社区版本）。
	3.安装CUDA9.2
	4.安装python 2.7.6以上版本（只限2.7版）
	5. 修改cuBitCrack.bat 文件：
	    cmd下运行 cudaInfo.exe 
	    ID 替换 -d 数值  （表示设备编号）
	    MP替换 -b 数值  （表示设备处理器数量）
	    Cores 替换 -p  （同一时间多少个计算点）
	6. 运行 python  python_cuCrackCheck.py
	   每运行1000亿次重新向服务器请求新的数据计算区间，
	   服务器计算区间已经过4年的计算产生大量的计算区间并做重新排序，看起来就像是随机分布计算区间。
	   找到地址后会存out.txt文件。
